    Brightcove Universityトレーニングコース

    このトピックでは、現在利用可能なブライトコーブ大学のトレーニングコースの一覧を示します。

    Video Cloud

    ビデオクラウドコース
    ビデオクラウドラーニングパスの概要 https://university.brightcove.com/catalog/learning-paths/33246

    このラーニングパスでは、Video Cloud Studioの使用を開始するために必要なすべてのツールを学びます。

    Video Cloud を初めて使用する場合やリフレッシャーが必要な場合は、基本についてはこちらから始めるのに最適な場所です。
    Video Cloud アカウントの使用を開始する https://university.brightcove.com/catalog/courses/920195

    このコースでは、アカウントの設定と管理を管理する方法を学びます。
    ブライトコーブ動画クラウドによるメディア管理 https://university.brightcove.com/catalog/courses/919975
    このコースでは、Video Cloud のアップロードモジュールとメディアモジュールを使用して、視聴者に最適なエクスペリエンスを提供するために、コンテンツを管理する方法を学びます。
    ブライトコーブプレーヤーの概要 https://university.brightcove.com/catalog/courses/841750

    このコースでは、次のことを学びます。

    1. Video Cloud で新しいプレーヤーを作成する方法
    2. プレーヤーの設定方法:
      • プレーヤーのプロパティを編集し、プレーヤーの外観をカスタマイズする
      • プラグインを追加する
      • プレイヤー設定ページの詳細
    3. コンテンツをプレビューして公開する方法
      • プレーヤーの変更をプレビューして公開する
      • 新しいプレーヤーでメディアモジュールでビデオを公開する
    タグでワークフローを簡素化 https://university.brightcove.com/catalog/courses/920020
    動画のワークフローを改善したいなら、このコースはあなたのためのコースです。ベストプラクティス、デモなど、ブライトコーブの幅広い製品スイート全体でタグを効果的に使用する方法を学びます。
    取り込みプロファイルの理解 https://university.brightcove.com/catalog/courses/920193
    ここでは、動画を Video Cloud にアップロードする際にどのように処理されるか、コンテンツに適した取り込みプロファイルを選択することの重要性、コンテンツのニーズに応じてカスタムプロファイルを作成する方法について説明します。
    カスタム分析レポートの作成とスケジュール設定 https://university.brightcove.com/catalog/courses/560005
    カスタム分析レポートでは、アナリティクスレポートに表示される内容を正確に定義し、1 回または日次、週次、または月単位のスケジュールで実行できます。レポートはCSV形式またはXLSX形式で利用でき、新しいレポートの通知を自分や他のユーザーに電子メールで送信することができます。このコースでは、カスタムレポートを作成および管理する方法について説明します。
    ビデオクラウド分析の概要 https://university.brightcove.com/catalog/courses/511393
    Video Cloud Analytics 入門トレーニングコースは、Video Cloud Analytics の使用に慣れるのに最適な方法です。このコースはVideo Cloudの使用を始めたばかりのパブリッシャー向けです。

    Brightcove Gallery

    Brightcove Gallery
    ギャラリーの紹介 https://university.brightcove.com/catalog/courses/504770
    この入門コースでは、ブライトコーブギャラリーで作成できる動画体験の種類や設定など、ブライトコーブギャラリーの概要を説明します。

    このコースを開始する前に、カタログにある「Video Cloud 入門コース」を完了することをお勧めします。

    このコースを修了すると、構築できるさまざまなタイプのエクスペリエンスに関する追加のトレーニングにご興味があるかもしれません。カタログには、「ギャラリーを使用したページ内エクスペリエンスの構築」および「ギャラリーを使用したポータルエクスペリエンスの構築」もあります。

    ギャラリーを使用したポータルエクスペリエンスの構築 https://university.brightcove.com/catalog/courses/504795
    このコースでは、Gallery Portal エクスペリエンスを作成、カスタマイズ、公開するプロセスについて説明します。ギャラリー設定、新しいエクスペリエンスの作成、サイトへのビデオの追加、サイトプロパティのカスタマイズ、ポータルエクスペリエンスのプレビューと公開の方法について説明します。

    このコースを開始する前に、Video Cloud の実用的な知識を持っていることをお勧めします。カタログには、Video Cloud 入門コースがあります。このコースは、開始する前に良い基盤となります。また、ギャラリーで何ができるのか概要をご希望の場合は、「ギャラリー・イントロダクション」コースをおすすめします。

    ギャラリーを使用したページ内エクスペリエンスの構築 https://university.brightcove.com/catalog/courses/504717
    このコースは、ギャラリーを初めて利用する出版社を対象に作られています。このコースを受講する前に、Video Cloud の実用的な知識を持っていることをお勧めします。開始する前に、Video Cloud 入門コースを受講してください。

    このコースでは、ビデオをビジネス継続性戦略に統合する方法を学びます。
    ブライトコーブキャンペーンで動画マーケティング戦略を最適化する https://university.brightcove.com/catalog/courses/757490
    ようこそ!このコースでは、ビデオの力を活用してリードをファネルを下げる方法と、動画分析を分析して改善する方法を学びます。

    Brightcove Audience

    Brightcove Audience
    ブライトコーブオーディエンスの紹介 https://university.brightcove.com/catalog/courses/513099
    オーディエンスモジュールは、オーディエンス対応の Brightcove Player 内で視聴された動画の動画エンゲージメントデータを追跡するために使用されます。表示データは Video Cloud に保存され、必要に応じて、Eloqua、Marketo、HubSpot、Salesforce などの一般的なマーケティング自動化プラットフォームに同期できます。

    ブライトコーブオーディエンスの概要では、Audience モジュールについて説明し、オーディエンスが提供するツールの使用方法について説明します。

    ブライトコーブソーシャル

    ブライトコーブソーシャル
    ブライトコーブソーシャンの概要 https://university.brightcove.com/catalog/courses/598128

    ブライトコーブソーシャルを使用すると、YouTube、Facebook、Twitter などの一般的なソーシャルメディアプラットフォームに Video Cloud アカウントで簡単に動画を公開できます。

    Brightcove Social の概要では、Social モジュールのさまざまな部分を理解し、ソーシャルメディアプラットフォームで動画を共有し始めるのに役立ちます。また、共有履歴を追跡し、場合によっては自動同期を設定できるため、各動画を手動で共有する必要がありません。また、さまざまなソーシャルプラットフォームで動画がどのように機能しているかに関する分析を確認することもできます。
