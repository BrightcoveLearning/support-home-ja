    Brightcove Systemのステータスページを表示する

    このトピックでは、Brightcove のサービスおよびアプリケーションの更新に関する [Brightcove システムステータス] ページを表示します。

    ブライトコーブは、すべての Brightcove サービスとアプリケーションのステータスを継続的に監視します。サービスに何らかの中断があると、システムステータスページに表示されます。本ページには、予定されているメンテナンス通知もすべて掲載されます。

    [システムステータス] ページにアクセスするには、 https://status.brightcove.com にアクセスするか、ナビゲーションヘッダーから [サポート] アイコン ( ) をクリックし、[  システムステータス ] をクリックします。

    投稿されたすべての通知には、以下の情報が含まれます。

    • インシデントステータス
      • 運用可能 (緑)
      • 通知 (青-システムは現在動作中ですが、遅延または断続的な停止の影響を受ける)
      • 部分的なサービス中断 (黄色)
      • フルサービスの中断 (赤)
    • コンポーネント － 影響を受けるサービスまたは領域
    • 場所
      • アジア・太平洋
      • アメリカ
      • エメア
      • 日本
    • 現在の状態とステータス。状態には以下があります。
      • 調査中-エンジニアが問題を調査中
      • 特定-エンジニアが問題を特定し、修正を検討している
      • モニタリング-問題の解決とエンジニアリング監視により、すべてが正常であることを確認する
      • 解決済み-問題が解決しました

    投稿されたメッセージはすべて英語で、使用時間は東部標準時 (GMT-5) です。

    システム状態の履歴を表示する

    システムステータスの履歴を表示するには、ページ上部の[履歴]リンクをクリックします。

    システム状態の変更を購読する

    サインアップして、システムステータスが変更されたときに通知を受け取ることができます。このセクションの構成は次のとおりです。

    • メールを購読中
    • webhookを購読中
    • サブスクリプションの管理

    メールを購読中

    メールで通知を受信するには、[ 購読 ] ボタンをクリックし、メールアドレスを入力して [ 購読 ] をクリックします。

    webhookを購読中

    ウェブフックは、HTTP POSTで作成された定義されたコールバックです。ステータスの更新がある場合に、コールバックが JSON オブジェクトをポストする URL を指定します。

    システムステータスが変更されたときに通知を受けるようにサインアップするには、[ 購読 ] ボタンを選択します。次に、電子メールアドレスとともに JSON ペイロードを受信する URL を入力します。

    購読すると、次のようなサブスクリプションの確認が表示されます。

    サブスクリプションの管理

    登録すると、「購読の管理」ページへのリンクを記載した電子メールが送られてきます。このページで、お客様の購読について管理できます。通知の受信を希望するコンポーネント/製品/リージョンを選択できます。項目を選択し、[購読の保存]をクリックします。

    購読を解除するには、[サブスクリプションの管理] ページの下部にある [ 登録解除 ] リンクをクリックします。
